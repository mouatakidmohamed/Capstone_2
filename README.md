# Project Overview

This project analyzes in-store sales data for **EmporiUm**, a growing student bookstore chain that sells tech gear, art supplies, textbooks, and other products across multiple regions.

As a data analyst, I was assigned two sales territories, Maine and Maryland, to compare and analyze. Both territories are part of the Northeast Region.

# Goals

- Compare sales performance across the two assigned territories
- Identify top-performing stores and customers
- Analyze revenue trends by product category
- Provide a data-driven marketing recommendation for next quarter
